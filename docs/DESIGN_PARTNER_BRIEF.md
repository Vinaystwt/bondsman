# Design Partner Brief

Bondsman is looking for Casper teams that want accountable agent execution for real financial workflows.

## Best Initial Partners

- RWA invoice or procurement platforms.
- Treasury tooling teams.
- x402 paid-service providers.
- Agent wallets and agent infrastructure teams.

## What A Partner Gets

- A deterministic policy template for bond sizing and challenge windows.
- A verifier design for objective fault evidence.
- Portable receipts suitable for audits and agent-to-agent integrations.
- MCP and HTTP integration paths.

## What Bondsman Needs

- A narrow workflow with objective pass/fail evidence.
- Known loss caps and acceptable bond coverage.
- A production custody model for operator keys.
- Agreement on whether the first integration is executable or a blueprint.
